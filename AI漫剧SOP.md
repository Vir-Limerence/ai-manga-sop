# AI漫剧制作完整SOP（零基础小白版）

> 适用对象：完全没有AI或视频制作经验的纯新手
> 目标：独立制作一部完整的AI漫剧（漫画+配音视频）

---

## 一、什么是AI漫剧？

AI漫剧是用AI工具生成的漫画风格视频/动画，本质是：
- **漫画图片** + **台词字幕** + **AI配音** = 漫剧视频

常见形式：竖屏9:16，适合抖音/快手/视频号

## 三、制作流程（5步法）

### 第1步：剧本创作

**目标**：把小说/故事改编成漫剧脚本

**操作步骤**：
1. 找一本小说（或自己写一段故事）
2. 打开 DeepSeek
3. 输入提示词：
   ```
   我要将以下小说片段改编成AI漫剧脚本：
   
   [粘贴你的小说内容]
   
   请帮我：
   1. 提炼核心剧情（100字内）
   2. 按场景划分每集内容（每集8-15个场景）
   3. 每个场景输出：画面描述、人物表情、台词
   
   格式示例：
   场景1：画面描述（人物+动作+场景）| 台词内容
   ```
4. 复制生成的脚本，保存备用

**输出**：分镜脚本（场景编号+画面描述+台词）

---

### 第2步：AI生成图片

**目标**：根据脚本生成漫画风格图片

**操作步骤**：

**2.1 注册Leonardo.ai**
1. 访问 https://leonardo.ai
2. 用Google账号注册
3. 每日免费150积分

**2.2 创建角色参考**
1. 在Leonardo点击「Preprocessing」-「Character Reference」
2. 描述主角外貌：
   ```
   年轻女性，大眼睛，长发披肩，穿蓝色衬衫，漫画脸，微笑
   ```
3. 生成3张不同角度
4. 保存一张最满意的作为参考图

**2.3 生成场景图片**
1. 在Prompt框输入场景描述：
   ```
   [画面描述], anime style, comic style, vibrant colors, detailed, 4k
   --ar 9:16 --s 750
   ```
2. 开启「Image to Image」
3. 上传角色参考图
4. Image Strength 设为 0.3-0.5
5. 点击生成
6. 不满意就重新生成，直到满意

**常用风格Prompt**：
- `anime style` - 动漫风
- `comic style` - 漫画风
- `manga style` - 日漫风
- `chinese comic style` - 国漫画风

**输出**：每个场景对应的漫画图片

---

### 第3步：AI生成视频（可选）

**目标**：让静态图片动起来

**方法A：Runway**
1. 注册 https://runwayml.com
2. 上传图片
3. 输入动作描述：
   - ❌ "人物动起来"
   - ✅ "人物抬起右手，手指远方"
4. 生成4秒视频
5. 免费125credits/月

**方法B：快手可灵**
1. 访问 https://klingai.com
2. 上传图片
3. 描述动作
4. 生成视频

**方法C：仅用图片+配音**（更简单）
- 不生成视频，只用静态图片+配音也能做漫剧
- 每张图停留3-5秒，配上台词

**输出**：动态视频片段（可选）

---

### 第4步：AI配音

**目标**：为每个场景配上声音

**操作步骤**：

**方法1：ElevenLabs（推荐）**
1. 注册 https://elevenlabs.io
2. 点击「Voice Lab」-「Add New」
3. 输入台词
4. 选择声音（推荐：Rachel、Arnold）
5. 点击生成，下载MP3

**方法2：剪映内置配音**
1. 打开CapCut
2. 导入图片
3. 点击「Text」-「Text to Speech」
4. 选择声音，生成配音

**台词处理技巧**：
- 口语化：把书面语改成口语
- DeepSeek可以帮助改写台词风格

**输出**：每个场景的配音文件

---

### 第5步：剪辑合成

**目标**：把图片+配音+字幕合成完整视频

**操作步骤**：
1. 打开 CapCut（剪映国际版）
2. 创建项目，比例 9:16（竖屏）
3. 导入所有图片，按顺序排列
4. 导入配音，对齐到对应图片
5. 调整每张图时长：3-5秒
6. 添加字幕：
   - 点击「Text」-「Auto Captions」
   - 选择配音语言，自动生成
   - 调整字体、大小、位置
7. 添加背景音乐：
   - 点击「Audio」-「Sounds」
   - 搜索免费音乐，添加
8. 预览，检查：
   - [ ] 画面与配音同步
   - [ ] 字幕正确
   - [ ] 背景音乐不抢戏
9. 导出：1080P或4K，MP4格式

**快捷键**：
- 空格：预览
- Ctrl+Z：撤销
- Ctrl+C/V：复制/粘贴

## 四、自检清单

完成制作后，检查以下几点：

- [ ] 角色外观一致（同一角色表情/服装是否统一）
- [ ] 画面清晰无明显瑕疵
- [ ] 配音清晰、情感合适
- [ ] 字幕与配音同步
- [ ] 画面时长合理（不拖沓）
- [ ] 背景音乐合适

## 五、常见问题FAQ

### Q1: 人物不一致怎么办？
A：
1. 用同一套Prompt生成
2. 使用Image to Image保持角色一致性
3. 用LoRA训练专属角色

### Q2: 画面有畸形/多手指怎么办？
A：
1. 加入Negative Prompt：
   ```
   deformed, bad anatomy, extra fingers, missing fingers
   ```
2. 降低Image Strength
3. 重新生成

### Q3: 视频动作不自然怎么办？
A：
1. 动作描述要具体（"抬手"而非"动"）
2. 缩短时长（3-4秒）
3. 用静态图片+配音代替动态

### Q4: 配音需要多少钱？
A：
- ElevenLabs免费：1000字符/月（约2-3分钟）
- 剪映免费：无限量
- 完全免费：Coqui TTS（本地运行）

### Q5: 素材侵权问题？
A：
- 选择商用授权的AI工具
- Leonardo付费版可商用
- 避免直接模仿特定艺术家风格
